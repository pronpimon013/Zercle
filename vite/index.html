Exercise 1: Development Server

A. What is the role of the index.html file?

It is the default file of a website. It is the first page that the web browser loads when accessing a folder or a website domain.

B. What is the URL you used to view the page? Is it HTTP or HTTPS?

It uses HTTP. The URL is: http://localhost:63342/vite/index.html?_ijt=alaoorr8l24kou72s6pmre5sh3&_ij_reload=RELOAD_ON_SAVE

C. What HTTP status code do you see for index.html? What does it mean?

The status code is 200 OK, which means the page was successfully loaded.

D. How can you tell the page is being served locally vs from a real domain?

You can tell it's being served locally by looking at the URL. If it includes localhost, 127.0.0.1, or starts with file:///, it means the page is running from your local machine.

E. What tools did you use to inspect the response?

Right-click on the page and select “Inspect”, then go to the “Network” tab to view the status code and response details.
